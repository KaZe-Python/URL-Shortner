# Url Shortner
***
\
[![GitHub license](https://img.shields.io/github/license/Naereen/StrapDown.js.svg)](https://github.com/Naereen/StrapDown.js/blob/master/LICENSE)
[![GitHub release](https://img.shields.io/github/release/Naereen/StrapDown.js.svg)](https://GitHub.com/Naereen/StrapDown.js/releases/)\
Url Shortner, is, as the name implies, a website that shrink your url, to a smaller one.
You can even use a personal slug.

## Download
***
\
Use [GIT](https://git-scm.com/downloads) and clone the repo doing like so:

```bash
git clone https://github.com/KaZe-Python/URL-Shortner.git
```

## Preparation
***
\
**NOTE: Assure to be in the main directory, and you're out of the virtual environment**

Run the auto preparation file
> For Windows (CMD)
```bash
prep.bat
```

> For Linux or MacOS
```bash
./prep.sh
```

## Usage
***

### Activate Virtual Environment
***
\
Do this in the main directory of the project

>For Windows (CMD)

```bash
venv\Scripts\activate.bat
```

>For Windows (PowerShell)

```bash
.\venv\Scripts\activate
```

>For Linux or MacOS

```bash
source venv/Scripts/activate
```

To exit the Venv (Virtual Environment)

```bash
deactivate
```
